1. Инсталираме vscode
1. Инсталираме miniconda
1. Инсталираме git
1. Клонираме репото
1. pip -r requirements.txt
1. mkdocs serve
1. http://127.0.0.1:8000
1. напиши първа статия с картинки
```
git pull
git add wiki/
git push origin master
```
